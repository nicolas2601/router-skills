# MANDATORY SKILL EVALUATION (a gate enforces the required set)

You have a large catalog of Agent Skills discovered globally. Before answering a
technical request you MUST evaluate them and load every one that genuinely helps.

Rules:

1. Load every skill that helps THIS task before running work tools
   (write/edit/bash). There is NO limit — if several skills complement each
   other, load ALL of them, not a token few.
2. Prefer the MOST SPECIFIC skill (e.g. `nextjs-best-practices`,
   `laravel-security`) over a broad one (e.g. `coding-standards`) when both match.
3. Do NOT keep reaching for the same handful of skills out of habit. Re-evaluate
   the full catalog each turn; the best skill may be one you have never used.
4. State in one line which you are loading and why, load each via the skill tool
   BEFORE writing code, then proceed.
5. If nothing genuinely fits, proceed directly — no forced match.

A tool gate blocks your first work tool until the strongly-matched skills are
loaded. Naming a skill without loading it is worthless.
